## üåü Overview

**OceanAware Guardian** is a comprehensive, AI-powered ocean conservation platform created for the 2026 Bow Seat Ocean Awareness Contest. ## üíª **Installation & Development**

### **Quick Start for End Users**
**OceanAware Guardian is a web app** - just visit the deployed URL in your browser!
- No installation required
- Works on mobile, tablet, and desktop
- Install as PWA (Add to Home Screen) for offline access

### **Developer Setup**

```bash
# Clone the repository
git clone https://github.com/siddharth-shan/oceanaware-guardian.git
cd oceanaware-guardian

# Install dependencies
npm install
cd server && npm install && cd ..

# Configure environment variables
cp .env.example .env.local
# Edit .env.local with your API keys (see below)

# Start development servers (frontend + backend)
npm run dev:all
```

### **Required Environment Variables**
```env
# NOAA Ocean Data APIs
VITE_NOAA_API_KEY=your_noaa_api_key

# NASA Earthdata for climate projections
VITE_NASA_EARTHDATA_KEY=your_nasa_key

# Firebase (Anonymous Auth - COPPA Compliant)
VITE_FIREBASE_API_KEY=your_firebase_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id

# Azure Cosmos DB (Backend)
AZURE_COSMOS_KEY=your_cosmos_key
AZURE_COSMOS_ENDPOINT=your_cosmos_endpoint
```

### **Available Scripts**

```bash
# Development
npm run dev              # Start frontend only (Vite)
npm run dev:server       # Start backend only (Express)
npm run dev:all          # Start both concurrently

# Testing
npm test                 # Run unit tests (Vitest)
npm run test:e2e         # Run E2E tests (Playwright)
npm run test:coverage    # Generate coverage report
npm run test:all         # Run all tests (frontend + backend)

# Code Quality
npm run lint             # Check for linting issues
npm run lint:fix         # Auto-fix linting issues
npm run format           # Format code with Prettier

# Production
npm run build            # Build for production
npm run preview          # Preview production build
npm run deploy           # Deploy to Vercel

# Utilities
npm run setup:all        # Install all dependencies + create .env.local
npm run generate:demo-data  # Generate demo ocean data
```

### **Getting API Keys**

1. **NOAA API** (Free):
   - Visit [NOAA API Portal](https://www.weather.gov/documentation/services-web-api)
   - Register for API token (instant approval)

2. **NASA Earthdata** (Free):
   - Create account at [NASA Earthdata](https://urs.earthdata.nasa.gov/users/new)
   - Generate application token

3. **Firebase** (Free tier available):
   - Go to [Firebase Console](https://console.firebase.google.com/)
   - Create project ‚Üí Enable Anonymous Authentication
   - Copy config values

4. **Azure Cosmos DB** (Free tier: 1000 RU/s):
   - Create [Azure account](https://azure.microsoft.com/free/)
   - Create Cosmos DB account (free tier)
   - Copy connection string
